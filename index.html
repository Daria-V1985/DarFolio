‹/› Petrograd bike

Этот проект представляет собой задание, данное на собеседовании.

Задача: сверстать лендинг с адаптивом по уже готовому и предоставленному макету из Figma, с использованием анимаций разной сложности.

Результат: часть анимаций была реализована с помощью JS-библиотеки GSAP. Адаптив был выполнен под все разрешения экранов отзывчивым способом через собственный сборщик проектов на gulp/webpack.

‹/› Burger House

Верстка продающего лендинга сети ресторанов Burger House.

Задача: сверстать лендинг с адаптивом по уже готовому и предоставленному макету из Figma, с реализацией возможности перерасчета стоимости продукции в зависимости от страны проживания + возможности при клике на кнопку "Заказать" автоматическое добавление в поле формы заказа названия товара.

Результат: лендинг адаптирован под все разрешения экранов отзывчивым способом. Все ссылки кликабельны. На JS реализован весь необходимый функционал. А также проработана валидация полей формы и добавлен параллакс-эффект.

HTML5 CSS3 SCSS Javascript ES6
‹/› Dorofeya

Адаптивная верстка Интернет магазина мыла ручной работы Dorofeya.

Задача: сверстать сайт интернет-магазина с адаптивом по готовому и предоставленному заказчиком макету Figma, с реализацией фильтра на странице каталога товаров, слайдера карточек товаров и переключение списков с помощью табов.

Результат: интернет-магазин адаптирован под все разрешения экранов. Все ссылки кликабельны. На JS реализован весь необходимый функционал без подключения корзины.
